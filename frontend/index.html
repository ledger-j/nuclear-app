Anomaly logic

The app should avoid false alarms from normal background swings, sensor outages, storms, or isolated media rumors.

1. Measurement layer Compare latest dose rate with local rolling baseline and nearby station median.
2. Context layer Check wind, rainfall, wildfire smoke, thunderstorm conditions, and known maintenance notices.
3. Narrative layer Cluster regulator updates, local news, and social/open signals around named plants.
4. Alert layer Escalate only when score exceeds threshold and at least two source families agree.
